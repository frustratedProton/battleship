# Battleship Game

A Battleship game built to learn **Socket.IO** real-time communication.

Built with **TypeScript**, **React**, **Tailwind CSS**, and **Socket.IO**.

## Features

- Single-player vs Smart AI (targets adjacent cells after hits)
- Real-time multiplayer
- Drag & drop ship placement

## Quick Start

```bash
# Install
npm install

# Add .env file
PORT=5173

# Run
npm run dev
```

## What I Learned

- Socket.IO event handling (emit/on)
- Real-time game state synchronization
- Room-based multiplayer architecture
- Managing turns between players

## Tech Stack

React • TypeScript • Tailwind CSS • Socket.IO • Vite • Vitest
